This is the folder which could be used in the benchmark codes.

### Installation

Install ```smplx``` from [smplx](https://github.com/vchoutas/smplx), and put it under this folder:

```
|-- data_processing
    |-- optimization
        |-- __init__.py
        |-- bvh2smplx.py
        |-- utils.py
    |-- utils
        |-- load_smplx_params.py
        |-- mesh.py
        |-- VTS_object.py
        ...
    |-- smplx
        |-- smplx
            |-- body_models.py
            |-- lbs.py
            ...
        ...
```
